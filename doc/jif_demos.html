Jif Demos

To show how to program in Jif, a collection of sample Jif programs are included in the jif-0.9/demo directory: 

Demo Purpose
Hello.jif To print out "Hello, the world!". 
cat.jif To print out the contents of a text file. This demo shows how to read from a file.
jiflib/jif/util jif.util package. This package contains a Jif implementation of Hashtable and LinkedList.
jiflib/jif/io jif.io package. Java programmers will see some familiar classes such as InputStream and OutputStream. In fact, this package is ported from the java.io package of GNU Classpath. This package shows how to use label parameters to port Java packages to Jif.
