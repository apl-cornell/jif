Jif: Java + information flow

Version 0.9.2, August 2001

This package contains the Jif compiler and the Polyglot extensible Java compiler toolkit.

Jif is a security-typed programming language that extends Java with support for static information flow control. It is being developed by the research group of Prof. Andrew Myers at Cornell University. Jif is written in Java and is built on top of the Polyglot extensible Java compiler toolkit.

The current version of Jif is available from http://www.cs.cornell.edu/jif

Installing Jif

Suppose you have already downloaded and unpacked the Jif Distribution. Your installation directory contains the following sub-directories:

  • src: the complete source code of the Jif compiler
  • lib: the Jif runtime libary files, the Polyglot library files and the Java Cup library
  • doc: the documentations
  • demo: Jif sample programs
  • images: logos

Prerequisites

To install Jif, we require the following:
  • JDK 1.2 must be installed and in your path. This includes the binaries: java, javac, javah, jar, javadoc, and rmic and the libraries tools.jar and rt.jar.
  • On Windows, you will need the Cygwin environment. Invoke bash and ensure your path is set up properly as above.

Installing the Jif compiler

Follow these steps to install the Jif compiler:
  1. Change directory to the top level directory of your unpacked Jif distribution, jif.
  2. Run "sh ./configure".
    3. Two new sub-directories will be created: bin and users. The bin directory contains scripts for running the Jif compiler. The users directory contains the jif.principal package, in which you can find the classes representing the principals on your local machines and some sample principals like Alice and Bob. See Create Principals for more information.
  3. Update the PATH environment variable to include the jif/bin directory. Or copy the files in the bin directory to a directory appearing in your PATH environment variable.

Now you can test your Jif compiler installation. Go to the demo directory and run:

   jifc Hello.jif
   jif Hello

If you see "Hello, the world!" printed on the screen, you have successfully installed Polyglot. Otherwise, make sure that you have followed every
instruction. If you still cannot run the Hello demo, please contact the authors.

To rebuild the Jif compiler, go to the src directory, run:

    make

The output directory for class files is src/classes. (On the Windows system, you will need a Microsoft C++ compiler to rebuild Jif.)
