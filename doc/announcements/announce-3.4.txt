We are pleased to announce release 3.4 of the Jif compiler at:

    http://www.cs.cornell.edu/jif

The new version of Jif includes several nice improvements:

    - More convenient handling of fatal errors
    - Automatic inference of the integrity of declassification 
    - Upgrade to Polyglot 2.5.1. This yields faster performance.
    - Hooks for supporting the Fabric distributed programming
      language, notably a label that represents the code provider.
